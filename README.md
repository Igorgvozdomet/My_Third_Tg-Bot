# Telegram Бот — Погода

Telegram-бот на Python, который показывает текущую погоду в любом городе с использованием OpenWeatherMap API.

## Функционал

| Команда / Действие | Что делает |
|-------------------|-------------|
| `/start` | Приветствие и инструкция |
| Ввод названия города | Показывает температуру, влажность, ветер, описание и отправляет картинку |

## Технологии

- Python
- pyTelegramBotAPI
- Requests
- OpenWeatherMap API
- JSON
